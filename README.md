# Task API - Spring Boot

Backend REST API desarrollada en Java con Spring Boot.

## Tecnologías
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Lombok

## Endpoints

### GET /tasks
Lista todas las tareas

### POST /tasks
Crea una tarea

### GET /tasks/{id}
Obtiene una tarea

### PUT /tasks/{id}
Actualiza una tarea

### DELETE /tasks/{id}
Elimina una tarea

## Cómo ejecutar

1. Clonar repo
2. Ejecutar `TaskApiApplication`
3. Acceder a `http://localhost:8080/tasks`
